# superstore-sales-dashboard
An interactive Power BI dashboard built using the Superstore dataset to analyze sales, profit, regional performance, and category trends. Includes data cleaning, KPI design, drill-down charts, and actionable insights for business decision-making.

## Features
- KPI cards for Sales, Profit, Quantity
- Category-wise and Segment-wise performance visuals
- Regional heatmaps and drill-down interactions
- Trend analysis with dynamic filters

## Tools Used
Power BI, Excel, Python (Pandas)

## Files
- Superstore_Sales_Dashboard.pbix
- superstore_dataset.csv
- Dashboard screenshots

## How to Use
Download the PBIX file and open it in Power BI Desktop.
